## Instalação

`useradd -m usuario;passwd usuario;usermod -a -G sudo usuario;chsh -s /bin/bash usuario`

```bash
git clone https://github.com/tonmarcondes/smartColeta.git
cd smartColeta
pip install -r requirements.txt
flask run
```
